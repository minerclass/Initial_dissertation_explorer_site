# Pedagogical Friction GitHub Pages Site

This folder contains a shareable static React site for the dissertation research plan.

## Publish on GitHub Pages

1. Push this repository to GitHub.
2. Open the repository on GitHub.
3. Go to **Settings** -> **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select your default branch and choose the `/docs` folder.
6. Save, then wait for GitHub to publish the site.

Your site URL will look like:

`https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/`

## Local preview

You can preview it locally by opening `docs/index.html` in a browser, or by serving the repository with a simple static server.

## Notes

- This setup does not require Node or a build step.
- React is loaded from a CDN, which keeps publishing simple.
- The original root files in this repository were left untouched.
